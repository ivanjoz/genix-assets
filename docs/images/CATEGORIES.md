# Store Image Categories

Each image lives in exactly **one** category folder, chosen by its primary subject/use.
Color, aspect ratio and lighting are NOT folders — they live in each folder's `IMAGES_LIST.md`.

Undocumented (optimized but not yet sorted) images sit at the root of `docs/images/`.
The documentation workflow moves each one into the correct category below.

| Folder | Use it for |
|--------|------------|
| `fashion-apparel` | Clothing, shoes, boutiques, racks, outfits, models wearing apparel |
| `jewelry-accessories` | Watches, rings, bags, belts, sunglasses, leather goods, scarves, hats |
| `beauty-cosmetics` | Skincare, makeup, perfume, haircare, nails, beauty product shots |
| `health-wellness` | Spa, self-care, supplements, meditation, medical, pharmacy |
| `fitness-sports` | Gym, yoga, running, sports gear, athletic activity & apparel |
| `food-beverage` | Cafés, restaurants, bakeries, prepared dishes, coffee, drinks |
